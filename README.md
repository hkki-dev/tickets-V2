# Cinema 예매 사이트 - 구현 기능 목록

## 공통 (모든 페이지)
- `data.js`를 `<script>` 태그로 로드 (fetch 미사용, 로컬 환경에서 바로 실행 가능)
- 첫 접속 시 localStorage에 영화/회원/예매 데이터 자동 초기화
- `data.js` 버전이 바뀌면 localStorage의 영화 데이터를 자동으로 최신화
- 로그인 상태에 따라 헤더 메뉴의 "로그인" 링크가 "로그아웃(이름)"으로 전환

## index.html (홈)
- 예매율(popularity) 기준 정렬 후 인기 영화 4개 동적 렌더링

## movies.html (영화 목록)
- 제목 검색 (입력창 + Enter 키)
- 장르별 필터링
- 검색어 + 장르 동시 필터링, 검색 결과 없을 때 안내 문구

## detail.html (상세)
- URL 쿼리스트링(`?id=`)으로 어떤 영화를 클릭했는지 전달받아 감독/출연/줄거리 등 동적 렌더링

## booking.html (좌석 선택)
- 영화/극장 select 동적 구성 (URL의 id로 영화 사전 선택)
- 상영시간 클릭 시 선택 표시(active) 전환
- 좌석 클릭으로 선택/해제 토글, 예매된 좌석은 클릭 무시
- 선택 좌석 / 인원 / 결제금액을 오른쪽 요약 패널에 실시간 반영
- "예매하기" 클릭 시 좌석 미선택이면 경고, 선택했으면 예매 내역을 localStorage에 저장 후 마이페이지로 이동

## login.html
- 아이디/비밀번호 미입력 검증
- localStorage 회원 정보와 대조해 로그인 성공/실패 처리
- 로그인 성공 시 세션 저장 후 홈으로 이동 (데모 계정: `user1` / `1234`)

## signup.html
- 필수 항목 미입력, 이메일 형식, 아이디 중복 검증
- 신규 회원 정보를 localStorage에 저장

## mypage.html
- localStorage에 저장된 예매 내역을 테이블로 동적 렌더링
- "취소" 클릭 시 확인창 후 해당 예매 삭제 및 목록 갱신

## admin.html
- 영화 등록 폼 → 유효성 검사 후 목록에 즉시 추가
- "수정" 클릭 시 prompt로 제목 / 상영시간 / 관람등급 수정
- "삭제" 클릭 시 확인 후 목록에서 제거
- 등록/수정/삭제 모두 localStorage에 반영되어 새로고침해도 유지
